# LAMP環境構築

## １．使用するOSやサーバ環境について
- OS：　Ubuntu　24.04.3 LTS ←　※課題２でバージョンが変わります
- 構築方法：　VirtualBox
- 参考資料：  
・Ubuntu公式ドキュメント（一次情報）  
・AI（CHATGPT、Jemini）： コマンドの意味や初学者向けの視点の解説に使用  
・Qiita： 環境構築の全体像を把握するための参考
- Ubuntu選定理由  
　自分が普段使用しているOS（Windows）以外に手を出すのは初めてだった為、初学者でも分かりやすいかつ、情報源の多い印象を受けたUbuntuを選択。  
　参考にしたwebサイト (https://qiita.com/free-honda/items/c8776286886b3ea31825)


## ２．OSのインストール


‐ Ubuntuのダウンロード、インストールを実施  
参考にしたwebサイトでは日本語対応版を使用していた為、同じく日本語対応版を使用  
課題１で記載したUbuntuとは異なるverのubuntu-ja-22.04 LTSを以後使用  
参考にしたwebサイト(https://webdesign-programming.com/virtualbox-ubuntu-install/)

#### 次回やってみたいこと

・今回使用したVirtualBox以外の選択肢としてWSL(Windows Subsystem for Linux)も使ってみたい

- ファイアウォールの有効化

## ４．Apacheのインストールとwebページが表示される環境作り  
- Apacheのバージョンは2.4系  
- 自動起動するように設定  
- サーバが再起動しても自動で起動するように設定
- VirtualHostの機能を利用し下記要件を満たす  
  ・ Webブラウザで`http://<自分の名字>-hbtask.local/` にアクセスすると「こんにちは。ありがとう。」と表示される  
  ・ Webブラウザで`http://<自分の名前>-hbtask.local/` にアクセスすると「ありがとう。こんにちは。」と表示される

## 課題を終えてみて  
完全未経験の私には、どれも初めて目にする内容ばかり。それどころか、このアウトプット用のgithubも初めて利用しています。  
アウトプットの書き方含めて最初はとても苦労しましたが、課題を進めていくうちに少しずつ内容が分かるようになってきて  
良い意味で、段々とITの沼に自分がハマっていくのを感じました。  

課題を開始してから２週間かかってしまい、もしかしたら進めるペースとしては遅かったかもしれません。  
ただ、自分にとってはとても新鮮で大きな学びになったと感じています。  

#### 具体的には  
そもそもLAMP、というものを初めて目の当たりにした私。  
Linux、Apache、MySQL、PHPという、開発環境のベースとなるものの頭文字を繋げているそう。  

Linux：初めてWindows以外のOSを使用した新鮮さ、ディストリビューションというものを知る。  
Apache：ブラウザを開く為の機能、ソフトの存在を知る。またこの機能一つとっても様々な仕様がある。  
MySQL：データベースを管理することが出来るソフト。今までの知見だとデータベース=Excel のようなイメージがあった。  
PHP：プログラミング言語として、名前は聞いたことはあったものの詳しい内容は知らなかった。今回触れることが出来て良かった。  

課題の前半部分は、本当に頭を抱えることが多く分からないことばかりでしたが、
課題が終わりに近づいていくにつれて、LAMPそれぞれの関連性になんとなく気が付き始めたり  
設定の仕方も、人によって様々なやり方があったり。色々な方面で知識を付けていくうちに、  
自分の理解度が上がっていくのを課題をやりながら感じとても楽しかったです。  
WordPressのブログ記事作成まで終えた時の達成感が本当にすごくて自分で驚きました。
